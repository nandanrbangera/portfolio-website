Data Analysis on Customer Churn Prediction

Utilized Python, Pandas, and NumPy to preprocess and analyze customer data, identifying key factors contributing to churn.

Applied machine learning algorithms using scikit-learn to build predictive models, achieving an accuracy rate of 85% in forecasting customer churn.

Created visualizations using Matplotlib to present insights and trends to stakeholders, aiding in the development of targeted retention strategies.
